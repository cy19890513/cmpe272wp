== Description ==

You can create wordpress user listing on front page. A search functionality will give your visitor more comfort in finding any user.
Visitor can also see more information by click on Visit info link and popup will show with admin selected fields.
Short code:[wp_user_searching]

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
